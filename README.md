A full-stack project built with Node.js, Express, MongoDB, Next.js, React, Redux, and PDF generation tools.
This project includes Authentication, File Uploads, PDF Creation, and a clean modern frontend.

⚙️ Tech Stack
Backend:-
Node.js,
Express.js,
MongoDB + Mongoose,
Multer (File Uploads),
BcryptJS,
Crypto,
PDFKit / pdf-creator-node ,
Nodemon ,
CORS ,
dotenv 

Frontend :-
Next.js ,
React ,
Redux Toolkit , 
Axios
